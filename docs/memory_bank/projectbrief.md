# Project Brief: Veterinary Clinic Website

## Core Goals
- Create a professional online presence for a small veterinary clinic.
- Provide information about clinic services, veterinarians, and contact details.
- Enable news updates and content management via an admin panel.
- Integrate a chatbot for basic AI-driven assistance.

## Scope
- **Frontend**: Responsive website with Home, News, About, and Contact pages.
- **Backend**: Django-based system for content management and form handling.
- **Features**: News listing/detail, veterinarian profiles, contact form, Yandex Maps integration.
- **Future**: AI Chatbot integration, PostgreSQL migration, Dockerization, and CI/CD.
